PROJECT: KS-DOS 0.1 PRE-ALPHA

[DESCRIPTION] KS-DOS is a hardened Linux distribution stripping away telemetry and bloat to provide absolute system sovereignty for Tech Haven Studios.

CORE FEATURES

  • KERNEL HARDENING: GRSEC/MGKFN patches to prevent buffer overflow/heap-spraying.
  • SILENT NETWORK: Forced WireGuard tunneling with encrypted community nodes.
  • MGK-INIT SYSTEM: C-based init replacing Systemd for reduced attack surface.
  • READ-ONLY ROOT: Core binaries mounted as RO, requiring hardware-key for override.
