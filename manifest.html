SYSTEM MANIFEST

Version History & Development Timeline

April 2026
KOS 0.2.9 + K-DOS 1.0 Beta

Major stability improvements and new beta releases across all systems.

NEWFIX
  • KOS: Experimental SMP scheduler for multi-core support
  • KOS: Rewrote page allocator with buddy system
  • K-DOS: Added cat, ps, kill commands to shell
  • K-DOS: Fixed command history buffer overflow
  • Desktop: Early window dragging support (unstable)
March 2026
KOS 0.2.5 + K-DOS 0.8 Beta

Hardware driver expansion and filesystem improvements.

NEW
  • KOS: Basic AHCI driver for SATA storage
  • KOS: PS/2 keyboard and mouse driver
  • K-DOS: Directory operations now functional
  • K-DOS: Basic file I/O with read/write
February 2026
K-DOS Desktop Alpha 0.1

First experimental GUI prototype released to alpha testers.

NEWBREAKING
  • Initial window manager prototype
  • Custom cursor rendering on framebuffer
  • Basic mouse input handling
January 2026
KOS 0.2.0 + K-DOS 0.5 Beta

Initial public beta release of both kernel and OS.

NEW
  • KOS: Boot sequence fully operational
  • KOS: Memory management with basic paging
  • K-DOS: Shell with ls, cd, help commands
  • K-DOS: RAM-based filesystem
December 2025
KOS 0.1 Alpha

First internal alpha build. Kernel boots in VirtualBox.

NEW
  • Minimal bootloader in Assembly
  • Basic VGA text mode output
  • Project officially started as BOOT AI
November 2025
Project Genesis

BOOT AI project conception and initial research phase.

NEW
  • Research into x86_64 architecture
  • Development environment setup (QEMU, GCC, NASM)
  • First "Hello World" bootloader compiled
